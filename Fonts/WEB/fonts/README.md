# Satoshi font files (TTF for Expo)

The Snapsy app loads Satoshi from this folder. Place these four files here:

- `Satoshi-Light.ttf`
- `Satoshi-Regular.ttf`
- `Satoshi-Medium.ttf`
- `Satoshi-Bold.ttf`

They are loaded in `app/_layout.tsx` via `expo-font`. If you have the full Fontshare package, copy the four files from your `Fonts/WEB/fonts/` (or equivalent) into this folder.
