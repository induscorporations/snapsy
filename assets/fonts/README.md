# Satoshi fonts (PRD design system)

Satoshi is loaded by the app from **`Fonts/WEB/fonts/`**. Ensure these four files are in that folder:

- `Satoshi-Light.ttf`
- `Satoshi-Regular.ttf`
- `Satoshi-Medium.ttf`
- `Satoshi-Bold.ttf`

Loading is enabled in `app/_layout.tsx` via `expo-font`. You can obtain Satoshi from [Fontshare](https://www.fontshare.com/fonts/satoshi); the Fonts folder in this repo may already contain the WEB package—copy the four .ttf files into `Fonts/WEB/fonts/` if needed.
